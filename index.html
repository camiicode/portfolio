Bootstraplus

Bootstraplus is a style library with the same Bootstrap writing syntax, which complements its version # 4 available.

This library allows you to make more advanced designs, and in many cases it allows us to have complete layouts, without touching a single drop of CSS, which makes it a plus, since it generates a list of completely reusable styles.

For example, this project is 95% built with Bootstrap 4 and Bootstraplus. Most of my layout work, I add my own library and that minimizes the hours of work that I dedicate to them without lowering quality.

This library, is made entirely with Sass and BEM methodology, this allows the library to come with all additional sub-labels for compatibility with all browsers.

Currently the library and documentation is under development in its BETA version, I must constantly test all the labels to verify their availability.
