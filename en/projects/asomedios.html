VERA The Voice of Truth

VERA, The Voice of Truth, is a revolutionary web application designed to combat misinformation in Colombian radio. This innovative project was developed by Nintera-Quadi for Asomedios, in collaboration with Colombia's five most influential radio stations: BluRadio, RCN Radio, Caracol Radio, W Radio, and other prominent radio networks in the country.

  2. Asomedios - VERA The Voice of Truth

Overview

The purpose of VERA is to validate the information broadcasted by the media, ensuring its truthfulness through a system that utilizes text-to-speech technologies combined with dynamic animations. The tool not only identifies false news but also offers clear and concise explanations about why certain information may be misleading, educating listeners and reinforcing trust in traditional media.

My Role

As part of the development team at Nintera-Quadi, I played a fundamental role in building this project. My main responsibilities included:

  • Design and development of the user interface (UI): Using HTML and CSS, I created an intuitive and attractive visual structure that enhanced the end-user experience.
  • System interactivity: I implemented interactive functions using JavaScript and TypeScript, ensuring smooth transitions and seamless integration with animations linked to the text-to-speech system.
  • Code management and architecture: I worked with Angular CLI as the main file manager, efficiently organizing the code and ensuring that the project met high technical quality standards.
  • Collaboration with multidisciplinary teams: I coordinated with designers, analysts, and other developers to ensure that each element of the software met the established objectives.

Additionally, I actively participated in the integration of technologies to ensure that the text-to-speech functionality operated accurately, providing clarity and dynamism to the explanations generated by the system.

Impact

The development of VERA had a significant impact on both the media and the audience:

  1. Strengthening trust: VERA helped reinforce the credibility of Colombia's major radio stations by providing listeners with tools to distinguish between true and false news.
  2. Education and awareness: Thanks to its educational functionality, the project promoted a more informed society, actively and effectively combating misinformation.
  3. Innovative collaboration: The union of Asomedios and the country's leading radio stations in a project of this magnitude set a precedent in the use of technologies to strengthen responsible journalism.
  4. Professional reputation: My work on this project allowed me to demonstrate advanced technical skills, as well as my ability to collaborate on projects of national impact.
